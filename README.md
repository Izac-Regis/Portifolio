# Izac Regis | Portfólio Pessoal 🚀

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar minhas habilidades, projetos e trajetória como Desenvolvedor de Software e estudante de Engenharia de Software.

## 🌟 Principais Funcionalidades

- **Design Moderno:** Utilização de **Glassmorphism**, paleta de cores escura com toques em neon, e um design totalmente focado em UI/UX.
- **Suporte Bilíngue (i18n):** Todo o conteúdo do site pode ser alternado instantaneamente entre Português (PT) e Inglês (EN) de forma nativa.
- **Totalmente Responsivo:** A interface se adapta perfeitamente em telas de celulares, tablets e monitores ultrawide.
- **Animações Fluídas:** Scroll suave, efeitos de *fade-in* nos elementos e um cursor brilhante personalizado que acompanha o movimento do mouse.

## 🛠️ Tecnologias Utilizadas

Este portfólio foi construído do zero focando em performance, sem a necessidade de frameworks pesados:

- **HTML5:** Estruturação semântica.
- **CSS3 Vanilla:** Estilização, animações, transições e variáveis CSS.
- **JavaScript Vanilla:** Lógica para o menu mobile, animações por scroll (IntersectionObserver) e motor de internacionalização (PT/EN).
- **Font Awesome:** Ícones vetoriais.
- **Google Fonts:** Tipografia (Outfit).

## 📁 Estrutura do Projeto

A organização dos diretórios segue as melhores práticas para aplicações frontend:

```text
📂 Portifolio/
 ├── 📂 assets/      # Imagens (perfil, logo) e o currículo em PDF
 ├── 📂 css/         # Arquivos de estilização (styles.css)
 ├── 📂 js/          # Lógica de interação e tradução (script.js)
 ├── 📄 index.html   # Arquivo principal do site
 └── 📄 README.md    # Documentação do projeto
```

## 🚀 Como Executar Localmente

Não é necessário rodar servidores complexos ou instalar dependências (como `npm install`). 

1. Clone o repositório para a sua máquina:
   ```bash
   git clone https://github.com/Izac-Regis/seu-repositorio.git
   ```
2. Abra a pasta do projeto.
3. Dê um duplo clique no arquivo `index.html` ou abra a pasta em uma extensão como o *Live Server* do VS Code.
